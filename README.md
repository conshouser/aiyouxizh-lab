# aiyouxizh-lab

> 一个用于归档和发布独立 HTML 页面的仓库。

## 简介

本仓库主要用于存放各类独立 HTML 页面及相关资源文件。这些页面可能来自不同的项目、实验或临时需求，统一整理在此处便于版本管理与分享。仓库不针对任何特定网站或域名，仅作为 HTML 内容的集中归档点。

## 仓库结构

```
aiyouxizh-lab/
├── pages/          # 存放各个独立 HTML 页面及其关联资源
├── assets/         # 公共资源文件（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录对应一个独立页面，包含 HTML 文件及所需的 CSS、JavaScript、字体、图片等资源。
- `assets/`：若多个页面共用同一资源，可放置于此，减少冗余。

## 页面归档说明

- 每个页面应具备独立的文件夹，命名简洁明确。
- 页面之间互不依赖，可单独打开运行。
- 若有外部依赖（如 CDN 库），请在页面内直接引用。
- 资源文件请尽量使用相对路径，便于本地或静态托管使用。

## 维护说明

- 新增页面：在 `pages/` 下创建新文件夹，放入 HTML 及相关文件，并更新本 README（可选）。
- 更新页面：直接修改对应文件夹内的文件，保持版本记录清晰。
- 废弃页面：可移入 `archived/` 目录（如存在），或直接删除，并在提交信息中注明。
- 欢迎提交 Issue 或 Pull Request 以协助维护。

## 使用方式

1. 克隆本仓库：`git clone https://github.com/your-username/aiyouxizh-lab.git`
2. 直接打开 `pages/` 下任意 HTML 文件即可在浏览器中查看。
3. 也可将仓库部署到静态托管服务（如 GitHub Pages）以在线访问。
